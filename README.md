
# Smart Expense Tracker

A simple and user-friendly Android application for managing daily expenses and tracking spending records.

## About the App

Smart Expense Tracker is an Android application designed to help users manage their daily expenses easily. Users can add, view, and track their spending records in a simple and organized way. The application provides a user-friendly interface for better expense management and financial tracking. This app is useful for students and individuals who want to maintain their daily budget efficiently.


## Features

- User-friendly interface
- Add daily expenses
- View expense records
- Expense management system
- Simple navigation
- Organized data display

## Technologies Used

- Java
- XML
- Android Studio
- Gradle
